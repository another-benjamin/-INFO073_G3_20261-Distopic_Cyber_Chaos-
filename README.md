Sobre el juego:

El juego se basa en el clasico snake pero reimaginado a la realidad distopica de Cyber Chaos.
  * Consigue las llaves para escapar de la ciudad.
  * Recoge baterías para poder seguir en pie.
  * Esquiva a los enemigos para no perder.
